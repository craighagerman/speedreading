This is a custom moodle plugin / module designed to set up speed reading tasks for ESL / EFL learners. 


INSTALLATION
=============
This module can be installed by (decompressing and) dropping into the 'mod' directory of a moodle installation. Then log in as an administrator and navigate to  Site Administration > Notifications. The module should automatically be detected and appear as ready to be installed. Click to install and it should be up and running. 

This module has been tested on Moodle version 2.4. It has not been tested on previous versions and due to changes in different versions may well not work on a previous version.
